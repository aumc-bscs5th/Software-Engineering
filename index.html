OBJECTIVES

  • To introduce the students to the principles and techniques of Software Engineering leading to better analysis and design
  • To enable students to get familiar themselves with state of the Art Software Engineering Construction Activities.
  • To get students equipped with Software Engineering Analysis, Design and Testing Techniques.
  • To introduce them to various structured modeling diagrams
  • To familiarize them with writing various Software Engineering documents

TEXTBOOKS

  • Software Engineering: A Practitioners Approach, 7th Edition Roger.S. Pressman
  • Software Engineering: Ivan Marsic, June 2009

Course Outline

This course provides an introduction to the fundamental concepts and methods of Software Engineering. Students shall be introduced to various software process models. Emphasis is on problem solving in systematic and disciplined manner using structured system modeling diagrams along with the required documentation throughout the system life cycle .Other issues like software project management, software testing, software risk management and software configuration management shall also be discussed
